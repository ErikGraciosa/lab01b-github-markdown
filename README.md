

# Procedure to clone repo and add directories
1) First, a repository was created via my github profile webpage.
1) Then, a link was generated on the github repository to enter into git bash.
1) Using git bash, the command `git clone <repository link>` was used to create the repository folder.
1) Change directory `cd` was used to move into the repo folder.
1) In the appropriate directories `mkdir` was used to create directories `src` and `assets`
1) `touch` was used to create the files in the appropriate directories.
    * `style.css`
    * `app.js`
    * `index.html`
    * `greeting.md`
